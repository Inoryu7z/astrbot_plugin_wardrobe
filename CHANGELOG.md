### v1.0.0

**👗 首次发布：图片衣柜管理插件**

**1. 📸 智能存图**

* 支持通过 `/存图` 指令或自然语言保存图片到衣柜库。
* 调用视觉模型自动分析图片内容，生成完整的结构化属性标签。
* 分析维度覆盖：分类、风格、服装类型、暴露程度、场景、氛围、姿势、构图、表情、动作风格等。
* 用户提供的描述强制写入 description 字段，确保意图不丢失。
* 支持图片格式自动检测（JPEG / PNG / GIF / WebP / BMP），保留原始格式。
* 图片大小限制可配置，默认 10MB。

**2. 🔍 语义检索**

* 支持自然语言检索图片，取图模型解析意图后从数据库匹配候选图片。
* 取图模型从候选中选出最匹配的图片，直接发送给用户。
* 无完全匹配时推荐最接近的结果。
* 候选数量和最大返回数量均可配置。

**3. 🤖 LLM 工具注册**

* `save_wardrobe_image`：保存图片到衣柜库，对话模型可自动调用。
* `search_wardrobe_image`：从衣柜库中搜索图片，对话模型可自动调用。

**4. ⚙️ 双模型配置与 Fallback**

* 存图模型和取图模型分别配置，各自支持主备切换。
* 主模型超时或失败时自动切换到备用模型。

**5. 🎮 管理指令**

* `/存图`：保存图片（管理员专用）。
* `/删图 <ID>`：删除图片并同步清理文件（管理员专用）。
* `/衣柜统计`：查看衣柜库统计信息。

**6. 📦 存储与安全**

* SQLite 数据库存储元数据，支持索引优化。
* 图片文件独立存储，删除时同步清理。
* 数据库更新操作使用字段白名单验证，防止 SQL 注入。
* 存图反馈开关可配置，开启后返回完整属性分析结果。
